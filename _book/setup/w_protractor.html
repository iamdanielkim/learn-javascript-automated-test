테스트 w/ Karma, Mocha

Karma는 테스트 환경을 제공할 뿐이고, 실제 테스트는 사용하는 test framework에 따라 작성 방법이 차이날 수 있다. 여기에서는 mocha, chai를 기반으로 테스트 작성하는 방법에 대해 설명한다.

실행

karma start karma.conf.js
